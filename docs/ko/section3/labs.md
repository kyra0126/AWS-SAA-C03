# AWS Console 및 서비스 둘러보기

## 1. 리전(Region) 선택

- 콘솔 우측 상단에서 현재 선택된 리전 확인 가능
- 지리적으로 가까운 리전을 선택하면 지연(latency)이 적음  
- 실제 위치와 달라도 원하는 리전 선택 가능
